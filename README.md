# finance-tracker
A personal finance tracker is a program the records the incone and expenses of a user and then caculates the remaining balance

Goal: Help user track their income, help user track their expenses, help user understand where their money goes, shows user remaining balance

What problem does it solve??

many people keep spending carelesslsy without knowing or keeping track of how their money was spent so this program solves that problem by allowing users see how their money is been spent and also keep track record of very single transcation which occured


project summary:

This project acts like a digital notebook for proper workflow and management of income or funds

Why does this project exist??

To help users make better decisions e.g instead of guessing or saying i think i have money, you can actually know the exact amount left or remaining


Who Benefits From It?
Students
Workers
Freelancers
Business owners
Investors
Families


Success Criteria

The project is successful if a user can:

Add income.
Add expenses.
View transaction history.
View current balance.
Understand their financial situation.


Why does each feature exist?

Add Income

Records money entering the system.

Add Expense

Records money leaving the system.

Show Balance

Tells the user how much money remains.

Show History

Allows users to review past transactions.
